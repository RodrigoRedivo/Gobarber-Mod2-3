## Startando a API

### Rodando o servidor

  ```$ yarn dev #Inicia o servidor```</br>
  ```$ yarn queue #Roda a fila```</br>

### Iniciando o container

  ```$ sudo docker start database```</br>
  ```$ sudo docker start mongobarber```</br>
  ```$ sudo docker start redisbarber```</br>
